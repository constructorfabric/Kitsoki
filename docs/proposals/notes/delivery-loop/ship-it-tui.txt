ship-it — TUI room renders (mockup; the `extends: "base"` typed-view target)
=============================================================================
These are hand-drawn renders of each ship-it room's View() for a single brief,
matching the kitsoki TUI frame (title bar / prose / kv / code / choice / status
line). They are the static-representation target slice 2's flows assert against.
Spatial-oracle note: each room is one fixed frame, so a layout regression
(e.g. the verbatim gate `code` block losing its border, or the choice list
re-ordering) is pixel-pinpointable by diffing this render against render_tui_png
output — the oracle anchors on the gate code-block and the choice rows.
